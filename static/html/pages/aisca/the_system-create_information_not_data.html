“Create Information, not Data”

With the system on, add, create data/information freely, in the format you prefer, and:

  • Be immediately alerted if there already is identical Information existing
  • Or let it join the Pool of Information, directly available to all, directly query-able and usable by all

Create and manage information data freely. Structured, or unstructured, in the format you wish, and let the system organize it, and turn it into directly accessible, networked Information with, more:

  • Automatic Hyper Linking between corresponding elements.
  • Automatic Appreciation (positive, negative...) of the Document, tunable according to given appreciation(s)
